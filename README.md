# The Effect of Occupation on Sleep, Physical Activity, and Overall Health

## Project Overview

This project explores how occupation is related to physical activity, sleep habits, stress levels, BMI, and sleep disorders.

The analysis was conducted using Microsoft Excel and is based on the Sleep Health and Lifestyle Dataset from Kaggle.

---

## Project Objectives

- Investigate how health indicators vary across occupations.
- Identify occupations with healthier and less healthy lifestyles.
- Explore relationships between physical activity, sleep quality, and stress.
- Present findings through an interactive Excel dashboard.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization
  ---

## Dataset

- **Source:** Sleep Health and Lifestyle Dataset (Kaggle)
- **Records:** 374
- **Original Variables:** 13
- **Additional Helper Columns:** 1 (Activity Level)

The dataset contains information about sleep duration, sleep quality, stress level, physical activity, BMI category, heart rate, blood pressure, daily steps, occupation, and sleep disorders.

---

## Research Questions

1. How does physical activity vary across occupations?
2. How does sleep duration vary across occupations?
3. How does stress level vary across occupations?
4. How does sleep quality vary across occupations?
5. How does BMI vary across occupations?
6. How does the prevalence of sleep disorders vary across occupations?
7. Is physical activity associated with sleep quality?
8. Is stress associated with sleep duration?
